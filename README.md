# Introduction to Python   
Introduction to the Python programming language and how Python is used in cybersecurity. Also explored the foundational Python concepts, including data types, variables, conditional statements, and iterative statements.

# Write effective Python code   
Expand ability to work with Python and its prebuilt and user-defined Python functions. Also explored how modules help provide access to reusable code, and focus on code readability.

# Work with strings and lists   
Explored more options for working with strings and lists in Python, and methods that can be applied to these data types. Applied this knowledge to wrote a short algorithm, and also used regular expressions to search for patterns in strings.

# Python in practice   
Put Python into practice and focused on automating cybersecurity-related tasks.   
Opening and reading files.   
Parse files and structure their contents.   
Strategies for debugging code.
